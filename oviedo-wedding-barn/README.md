# fresh project

### Usage

Start the project:

```
deno task start
```

This will watch the project directory and restart as necessary.

---
### Sitemap tool
https://deno.land/x/fresh_seo@0.2.1
```
# Run in project root

deno run            \
    --allow-read    \
    --allow-write   \
    https://deno.land/x/fresh_seo/init.ts
```
